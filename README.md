# LaraVuetufy

Laravel 5.8 + Vue 2 + Vuetify 2 Starter Template

### Installing
* git clone https://github.com/avgkudey/LaraVuetify.git
* cd LaraVuetify
* composer install
* npm install
* php artisan serve
* npm run dev - on separate terminal
* npm run watch
